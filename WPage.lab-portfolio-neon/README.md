# WPage.lab — Portfólio (HTML estático, Neon)
Pronto para deploy com **neon accents**.

## Arquivos
- `index.html` — site completo (neon + Tailwind CDN).
- `assets/logo.svg` — marca minimalista neon.
- `README.md` — instruções.

## Publicação rápida
**Vercel**: New Project → Import → Upload → selecione o .zip → Deploy.  
**Netlify**: Add new site → Deploy manually → arraste a pasta.  
**GitHub Pages**: ative Settings → Pages (branch main, pasta `/`).

## Personalização
- **WhatsApp**: edite `const telefone` no `index.html`.
- **Preços**: textos na seção *Serviços*.
- **Cores**: troque `--brand` e `--neon` no `<style>`.
